# Wallet API
> Сервис для управления кошельками пользователей
> Реализован на **FastAPI, PostgreSQL, SQLAlchemy (async)*** с поддержкой конкурентных операций и миграция Alembic

## Установка и запуск
1. Создайте .env файл ``` cp .env.sample .env ```
2. Установите зависимости ``` pip install requirements.txt ```
3. Запустите соберите и запустите контейнер приложения ``` docker-compose up --build ```
4. Запуск тестов ``` pytest -v ```

## Основные эндпоинты
| Метод  | URL                         | Описание                                   |
| ------ | --------------------------- | ------------------------------------------ |
| `POST` | `/create`                   | Создать новый кошелёк                      |
| `GET`  | `/{wallet_uuid}`            | Получить баланс по UUID                    |
| `POST` | `{wallet_uuid}/operation`   | Провести операцию `DEPOSIT` или `WITHDRAW` |

В ходе разработки принял решение о создании дополнительной ручки (```/create```), которая создаёт новый кошелёк
с нулевым балансом
